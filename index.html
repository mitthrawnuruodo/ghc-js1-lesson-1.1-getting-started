Lesson 1.1 - Getting Started

Open Console to see output from JavaScript:

  • Chrome:
    F12 or Ctrl + Shift + J (Cmd + Option + J on Mac) or click on the action button in the top-right corner and go to More Tools > Developer Tools
  • Microsoft Edge:
    F12 (or by clicking on the action button (top-left corner) > More tools > Developer Tools)
  • Mozilla Firefox:
    Ctrl + Shift + J (or Cmd + Shift + J on a Mac) or open it from the action menu â€“ By clicking on the action menu > Web Developer > Browser Console.
  • Safari:
    Cmd + option + C or open it by accessing the Develop tab at the top and clicking on Show Error Console. (NOTE: Developer mode must be activated in Safari preferences > Advanced by checking Show Develop menu in the menu bar)
